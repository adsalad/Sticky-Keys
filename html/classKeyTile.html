Detailed Description

This class represents the KeyTile object.

Author
Jose Torrealba

This class represents the KeyTile object. In our game, these are the three tiles you see hovering in the top left view of your screen. Each of these tiles represent a particular direction/movement option for the player. While in their default position, the player can move freely around the game. However, if an extra platform is needed for a puzzle, the user can click on these keyTile platforms and place them into the game world, thus being able to use them for extra mobility. However, this now locks the corresponding movement option for the player. This class contains the implementation of said concept below.
