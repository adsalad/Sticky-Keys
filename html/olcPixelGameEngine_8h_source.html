14  What is this?
15  ~~~~~~~~~~~~~
16  olc::PixelGameEngine is a single file, cross platform graphics and userinput
17  framework used for games, visualisations, algorithm exploration and learning.
18  It was developed by YouTuber "javidx9" as an assistive tool for many of his
19  videos. The goal of this project is to provide high speed graphics with
20  minimal project setup complexity, to encourage new programmers, younger people,
21  and anyone else that wants to make fun things.
22 
23  However, olc::PixelGameEngine is not a toy! It is a powerful and fast utility
24  capable of delivering high resolution, high speed, high quality applications
25  which behave the same way regardless of the operating system or platform.
26 
27  This file provides the core utility set of the olc::PixelGameEngine, including
28  window creation, keyboard/mouse input, main game thread, timing, pixel drawing
29  routines, image/sprite loading and drawing routines, and a bunch of utility
30  types to make rapid development of games/visualisations possible.
