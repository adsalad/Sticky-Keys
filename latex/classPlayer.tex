\doxysubsection{Detailed Description}
This class represents the playable character object. 

\begin{DoxyAuthor}{Authors}
Jose Torrealba, Cole Chang
\end{DoxyAuthor}
This class represents the player object. There will only be one of these that exist in a game at any time as this object is what the user will control. Through keyboard input (using W\+A\+SD, arrow keys and spacebar), the user can control the movement and position of this player character throughout the game. This class deals with the various input and corresponding changes to the player position, velocity and death/win status throughout the game.
